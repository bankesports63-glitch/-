SSD     

 SSD คือสื่อบันทึกข้อมูลที่แตกต่างจาก HDD ตรงที่ใช้หน่วยความจำแบบไม่เลือนหาย (แฟลช)
เพื่อจัดเก็บและสืบค้นข้อมูลกล่าวคือไม่มีชิ้นส่วนกลไกที่เคลื่อนที่ ซึ่งถือเป็นข้อแตกต่างที่สำคัญ

     RAM     

 RAM เป็นคำย่อของคอมพิวเตอร์ทั่วไปที่ย่อมาจาก Random-Access Memory (หน่วยความจำการเข้าถึงแบบสุ่ม) บางครั้งเรียกว่าหน่วยความจำพีซีหรือสั้น ๆ
ว่าหน่วยความจำ โดยหลักแล้ว RAM คือหน่วยความจำระยะสั้นของคอมพิวเตอร์หรือแล็ปท็อปของคุณ เป็นที่จัดเก็บข้อมูลที่โปรเซสเซอร์คอมพิวเตอร์
ของคุณต้องมีเพื่อเรียกใช้แอปพลิเคชันและเปิดไฟล์

     M.2     

 M.2 เป็นหน่วยความจำที่กำลังได้รับความนิยมในปัจจุบัน ด้วยขนาดที่เล็กและติดตั้งง่าย เพียงเชื่อมต่อกับพอร์ต M.2 และยึดด้วยน็อตเข้ากับเคสก็สามารถใช้งานได้ทันที
ซึ่งจุดเด่นของ M.2 จะเป็นการช่วยประหยัดพื้นที่ภายในเครื่อง และมีความเร็วสูงถึง 3,500 Mbps

     usbflashdrive     

 ไดร์ฟ USB หรือที่เรียกกันว่าแฟลชไดร์ฟหรือเมมโมรี่สติ๊กคืออุปกรณ์พกพาขนาดเล็กสำหรับเสียบต่อเข้ากับพอร์ต USB ที่คอมพิวเตอร์ของคุณ
โดยปกติไดร์ฟ USB มักใช้ในการจัดเก็บข้อมูล สำรองข้อมูล และถ่ายโอนไฟล์ระหว่างอุปกรณ์ต่าง ๆ
